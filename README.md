# SRS Rereading App (MVP)

A desktop-style single-page web app for scheduled spaced rereading with:

- Study Queue
- Sources list
- Source Portal
- Settings shell page

It supports state restoration, queue management, review timer/notes, audit-friendly review revisions, and mock viewer navigation.

## Run

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173`.
